# 🧬 Breast Cancer Predictor

[![Streamlit App](https://img.shields.io/badge/Launch%20App-Streamlit-ff4b4b?style=for-the-badge&logo=streamlit&logoColor=white)](https://breastcancerpredictapp.streamlit.app/)
